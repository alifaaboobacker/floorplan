# Project Setup

## Backend Setup
1. Navigate to the backend folder:
   ```
   cd Floor-Backend
   ```
2. Install dependencies: 
    ```
   npm install
   ```
3. Start the backend server:
   ```
   npm run dev
   ```
## Front end
1. Navigate to front end folder
     ```
     cd floor-frontend
     ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the frontend server:
  ```
    npm start
  ```
